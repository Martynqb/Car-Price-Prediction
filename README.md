# Car Price Prediction 🚗

## 📌 Project Overview
This project predicts the selling price of cars using machine learning.

## 🎯 Objective
To build a model that estimates car price based on features like year, price, fuel type, etc.

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn

## 📂 Dataset
Custom dataset with car details like:
- Year
- Present Price
- Kilometers Driven
- Fuel Type
- Transmission

## 🤖 Model Used
Linear Regression

## 📊 Steps
1. Data Loading
2. Data Preprocessing (Encoding)
3. Train-Test Split
4. Model Training
5. Prediction

## 📈 Result
Model predicts car prices with good accuracy using Linear Regression.
